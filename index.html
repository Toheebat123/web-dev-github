HTML & CSS Basics Summary

HTML Summary

HTML (Hypertext Markup Language) is used to define a page content, structure and meaning. It's not used to style a webpage. CSS is used for that.

  • HTML uses "elements" to describe (annotate) content.
  • HTML elements have an opening tag, content and closing tag.
  • You can also have void empty elements like images.
  • Elements can also be configured with attributes.
  • We have a long list of HTML available elements.

Learn more about available HTML elements on the MDN HTML element reference.

CSS Summary

CSS (Cascading Styles Sheets) is used for styling your page content.

  • Styles are assigned via property-value pairs.
  • You can assign styles via the "style" attributes.
  • To avoid code duplication, you typically use global styles (via the "style" element) instead.
  • Alternatively, you can work with external stylesheet file which you "link" to
  • When working with CSS, concepts like "inheritance", "specificity" and the box model should be understood.

    • Learn more about available CSS properties and values on the MDN CSS properties reference.
